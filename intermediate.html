                adding links
            

            
    <a href = "#">
    Click Here!
    </a>
              
            
            
'a href' is how you link something to another website or page.
I added the '#' because it isn't linking to anything just yet but in place is where you would add the link 
to the website or page location on your computer. After it you can add the name of the link. I chose to call
it 'Click Here!'.
            

            
    <style>
    a{
        color: black;
        background-color: white;
        text-decoration: none;
    }
    </style>
            
            
The colour of the writing will be black but the surrounding will be white.
The 'text-decoration' is the blue colour and outline that hyperlinks usually have, so by removing it I am free to add
my own text decorations. 
            

        
            

                Classes:
            

            
    <p class = "Welcome">
    Hey, Welcome!
    </p>
    
            
            
By adding  an element before the word class you have assigned that element
a name (or class) and you can edit it in CSS styles
            

            
    <style>
    .Welcome{
        color: pink;
    }
    </style>
            
            
The full stop before the name of the class shows that you are 
referring to a class in CSS and the name after specifies which one.
            

        
            

                Id's
            

            
    <p id = "step2">
    Hey, you are officially a coder
    </p>
              
            
            
By adding  an element before the word 'id' you have assigned that element
a name (or id) and you can edit it in CSS styles
            

            
    <style>
    #step2{
        color: red;
    }
    </style>
            
            
The hashtag before the name of the id shows that you are 
referring to a id in CSS and the name after specifies which one.
            

        
            

                Sectioning
            

            
    <div class = "top">
    <p>this is the top of the page<p>
    </div>
    
            
            
div is a good way of sectioning the program so that
everything you need to edit is in it's own space and to preven confusion. Make sure to used
meaningful names when naming the div class or the div id so that anyone else reading the code
understands it too
            

            
    <style>
    .top{
        background-color: orange;
        padding-bottom: 4vh;
    }
    </style>
            
            
Just like the first section because I made a div class I 
will use a full stop (.) reference the div. If i used id instead of class I would use a hashtag (#).
Padding is a functionality of CSS and it means how big of an area the decoration covers.
Padding has top,bottom,left and right. Just write it in the form: padding-(direction). it can be in any units
even percentages.
            

        
            

                Navigation bar
            

            
    <div class = "navbar">

    <div id = "brand">
        <h3>
            Hack.it()
        </h3>
    </div>

    <div id = "menu">
        <a href = "#">Home</a>
        <a href = "#">About us</a>
    </div>
    </div>
              
            
            
Using all the skills we have learnt we can make a navigation bar.
The 'brand' div is where we write the title of a website and the menu is the different navigation options.
You can make as many or as little as you want and design it however you like but remember to keep 'class' 
and 'id' names meaningful so that it's understandable and readable.
            

            
        <style>
        .navbar{
            display:flex;
            background-color: purple;
        }
        #brand{
            font-size:4vh;
            color: white;
        }
        #menu{

            margin-left:50%;
            margin-top:3%;
            color: #f2f2f2;
            text-align: left;
            text-decoration: none;
            font-size: 17px;
            }
        a{
            
            text-decoration:none;
            color: white;
            }
        </style>
            
            
I am using the name of colours but hex codes and rgb values
work as well. 'display: flex' just means that it will auto-align with the width of the page.
'Font-size' indicates the size of the font and can be measured by an units and percentages.
Margins are the distance between objects and the specified edge. There is bottom,top,right and left 
and it can be written as: margin-(direction).
            

        
            

                Footer
            

            
        <div class = "footer">
        <h3>
            Hack.it()
            this is the footer
        </h3>
        </div>
        
            
            
Using all the skills we have learnt we can make a footer.
The 'footer' div is where we write the content of the footer this can include links to social medias (using the method learnt at the top).
Remember to keep 'class' and 'id' names meaningful so that it's understandable and readable.
            

            
                <style>
        .footer{
            position: fixed;
            left: 0;
            bottom: 0;
            width: 100%;
            background-color: red;
            color: white;
            text-align: center;
        }
        </style>
            
            
I am using the name of colours but hex codes and rgb values
work as well. 'position:fixed' just means that it will stay in that position. There are 
different types of position attributes that you should feel free to explore and that we will explore
in later pages of the website.
