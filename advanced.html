Basic JavaScript

        
<p id= 'Greetings'> </p>
<script>
document.getElementById("Greetings").innerHTML="hello";
</script>

<script> is a tag that references JavaScript code. In this script I have named my written a word to be displayed by my HTML. 'document.getElementById("Greetings").innerHTML' is a function of JavaScript. It means that it should get the word/element that is referenced by the id given. My 'id' in this is 'Greetings' because that is what I called it in my HTML. By assigning 'document.getElementById("Greetings").innerHTML' to 'hello' I am saying that the element associated with the id referenced should be written/overwritten as 'hello' 

        
Getting user input

        
<form>
    <input type="text" id="input_id" value="">
    <button value="Submit" onclick="userinput()"> <button/>
    <div id="alert"></div>
</form>
<script>
    function userinput() {
        var variable = document.getElementById('input_id').value;
        document.getElementById('alert').innerHTML = 'The user input is: ' + variable;
    }
</script>
        
        
 Copy & Paste the code above!!   ↑

the tag >form> is a function of HTML it creates a form which is where you type whatever you want or is appropriate in the text field provided. The 'id' of the input is "input_id" which we refer to in our JavaScript below it. I have created a function called 'userinput' which is activated 'onclick' of the submit button in our HTML. There are many diffent modes but onclick tells the function 'userinput' to run when the button is 'clicked'.

The code between the <script> tag is JavaScript (commonly abbreviated JS): 'var variable' - this creates a variable and calls it 'variable' it can be any name you want I simply named it variable for teaching sake. document.getElementById('input_id').value- this gets the value/element associated with the referenced id. 'The user input is: ' + variable - I have concatenated a string and my variable. Concatenation is simply putting strings together. 'Strings' is a character type which is shown by "" or ''. 

        
External style sheets

        
We have seen when we use internal stlye sheets
with <style> tags but you can also have external style sheets.
we can reference this in the header. 

        
    <head>
        <link rel="stylesheet" href="*stylesheetname.css*">
    </head>
        
        
 This is how you reference an external style sheet.
The style sheet name can bee anything and the '.css' at the end 
is the file type that shows it's a CSS file.The CSS file is structured 
the same as the code inside the >style> tags.
